Lets Install Python

Installing on Windows

To begin, Click Here

Once you are there download the latest Windows installer for Python 3. Note: there will be different versions of Python 3, it does not matter which one you choose as long as you download Python3...

Installing on Mac OS X

Windows Tips

If you're on windows, you can start coding using Notepad.

If this is the case, you can open Notepad, type: 

print("Hello World")

Now that you have some code, lets save it and run it. You can go to File, Save As, save as helloworld.py.

Now to run the file, go to your command prompt, you can type in cmd in the windows search.

Once it opens, you want to make sure you are in the directory of where your python file is saved. For easy use, save it to your desktop.

Next you type the following: 

cd Desktop --> this goes to the directory where the python file is stored.
python3 helloworld.py --> this will execute the python code.

Your output should be: 

Hello World

There are many more advanced ways to do this. For now lets stick with simpicity.

Mac OS X Tips

If you're on Mac, you can start coding by using the given txt editor.

If this is the case, you can open the text editor, type: 

print("Hello World")

Now that you have some code, lets save it and run it. You can go to File, Save As, save as helloworld.py.

It might ask you if you want to overrite the .txt ending, choose yes and choose .py extension instead.

Now to run the file, go to your terminal, you can find this by hitting command and space then typing terminal.

Next you type the following: 

cd Desktop --> this goes to the directory where the python file is stored.
python3 helloworld.py --> this will execute the python code.

Calulating using Python

How would you find the product of 3 x 9.8 numbers? You can just use a calculator or a pencil.

The simplicity of that is the same using python. In our code editor lets type the following:

print(3 * 9.8)

Output will be: 29.4

Do you see a difference between what coded and what you would write? The difference is the 'x'.

In python, you will come to see that syntax is very important. That is why for multiplication we use the asterik (*) instead of the standard (x).

Solving a word Example:

I will give you 10 dollars everytime you walk my dog, and at the end of the month I will add 5 dollars. Lets say you do this everyday for a year, how much money will you have earned?

On paper the equations might look like:

$10 x 365 = 3650 --> Walking my dog everyday
$5 x 12 = 60 --> 5 dollars at the end of the every month
$3650 + $60 = $3710 --> Total money earned at the end of the year.

Luckily for python, you can do this directly as well. However you would not hav eto do any of the math, the program will do it all for you.

This is how it would look like if you wrote it in python:

print(10 * 365) --> OUTPUT: 3650
print(5 x 12) --> OUTPUT: 60
print(3650 + 60) --> OUTPUT: 3710

This is a very simple math operation. We will get much more into depth further down.

Python allows for the use of multiplication, division, addition, and subtraction. Though python is simple, it allows for much more advanced mathematical operations that we will not cover right now.

Python Operators

+ Addition

- Subtraction

* Multiplication

/ Division, when writing on paper, the most common form of division is ÷, but for python we use the forward slash.

Order of Operations

Just like doing math on paper, there are operations and logic that needs to be followed. Using PEMDAS is the best way to follow your logic when doing operations in python.

When doing math operations in python, the best way to follow logic is to use ().

Multiplication and division have a higher order than addition and subtraction, which means that they’re performed first. With this being said, if you create an equation in Python, multiplication or division is performed first, then addition or subtraction.

Example with no ():

Type the following in your text editor:

5 + 30 * 20

After running it, result will be 605

Example with ()

Using the same numbers as above. Type the following:

(5 + 30) * 20

Notice that the result is different. This time we get 700.

That is because we are saying: Do the operation in the parentheses first, then do the operations outside the parentheses.

Remember, multiplication and division always go before addition and subtraction. Unless () are used to control the order of operations

Variables

In programming, the term variable that is used to store information- numbers, text, lists of numbers and text, etc..

Example:

To create a variable named number, we will use the symbol =

We then tell Python the information we want it to hold.

In this case, we want our variable number to hold the number 25

So we type the following in our code editor:

number = 25

To show what information this variable holds, we can also add a print statement.

print(number)

After running it, out put should be: 25

In python we can also use more than one variable for the same item.

For example:

number = 25
new_number = number
print(new_number)
OUTPUT: 25

Here we see that we assign number, which holds the value 25, to our new variable, new_number. We can then print our new variable, and it will have the output of the old variable.

Using Strings in Python

What is a string?
When programming, string is a mmore technical term for text.

If you look at a book, the whole think can be a string, your name can be considered a string as well.
The most popular string phrase in python: "Hello World." Which you learned how to do in lesson 1.


Creating Strings

To create strings in python, you can use single quotes ('') and double quotes ("").

For example:

name = "Python" #or name = 'Python'

Now, if you run this, the output will be empty. Why? Because we need to access the variable and print it.

print(name) #OUTPUT: Python

When using strings, it is always wise that you look it over and make sure that you have closed off the string.
By this, I mean make sure there is " or ' at the ened of your string, depending on which one you are using.

If you don't you can run into some errors:
name = 'Python
print(name)
OUTPUT: SyntaxError: EOL while scanning string literal

This means that you did not close off your string, and if you look at the example, we missed a '.
One error that you will come by often is a SYNTAX error. This means that there was something done in your python program, that Python was not expecting, or Python was expecting something that you missed.
EOL means end-of-line, so the rest of the error message is telling you that Python hit the end of the line and did not find a double quote or single quote to close the string that you created.

    Basics of Lists in Python

    Lets say you are going to go to the grocery store to by: Eggs, Ham, Cheese, Bread, Lettuce, Tomatoes.

    We would be able to store this grocery list in python as follows:

    grocery_list = 'Eggs, Ham, Cheese, Bread, Lettuce, Tomatoes'
    print(grocery_list)
    OUTPUT: Eggs, Ham, Cheese, Bread, Lettuce, Tomatoes

    Though this is simple, we will not be able to manipulate this string for future use.
    Instead we can use a list.

    We write our new grocery list as follows:
    store_list = ['Eggs', 'Ham', 'Cheese', 'Bread', 'Lettuce', 'Tomatoes']
    print(store_list)
    OUTPUT:['Eggs', 'Ham', 'Cheese', 'Bread', 'Lettuce', 'Tomatoes']
    It does take a little more typing to create a list, but now we can access out items individually and manipulate our grocery list.

    For example, if we wanted to get the our 3rd item from our grocery list, we do this by entering the item position in the list (called the index position) inside square brackets ([]). Like this:
    print(store_list[2])
    OUTPUT: Cheese

    Now you might be asking, why did I put 2 instead of 3 since I wanted to get the 3rd item?
    This is because ists start at index position 0. The first item in a list is 0, then the second is 1, the third is 2, and so on.
    This may take some getting use to, but for computers, that is how it makes sense.
